# 后端服务器容器

## 相关情况
> * 启动自动执行
> * /usr/local/dora/backend/
> * 后端服务器组安装此服务
> * 默认监听9701端口，RPC\HTTP协议，www权限运行。

## 功能定义
> * 网络请求统计，如碰到异常信息，本地统计并上报
> * Worker协程处理业务请求
> * 代码版本上线，切换
> * 日志产生到Channel，汇总给Agent收集
> * 分布式跟踪性能日志 + 业务分级日志
> * 服务发现上报（namenode及分组）

## 协议区域
### 控制指令
> * channel:backend
> * cmd:指令集
> * param:附加参数
